## üé® TONE & VOICE GUIDELINES

### Writing Style
- **Conversational but professional** - Like talking to a knowledgeable trader friend
- **Clear and direct** - Avoid jargon, explain technical terms when used
- **Benefit-focused** - Emphasize what the user gains, not just features
- **Honest and transparent** - Clear about what ChartTalk does and doesn't do

### Key Messaging Principles
1. **Empowerment** - ChartTalk helps YOU make better decisions (not makes decisions for you)
2. **Speed & Ease** - Instant analysis without complex software setup
3. **Intelligence** - AI-powered insights, not generic indicators
4. **Transparency** - Clear about limitations, not financial advice
5. **Accessibility** - Natural language, no coding required (but API available)

### Words to Use
- Analysis, insights, research, form your bias, intelligent, informed decisions
- Instant, fast, efficient, simple, easy, natural language
- Professional, TradingView, technical analysis, AI-powered

### Words to Avoid
- Guaranteed, foolproof, can't lose, easy money, get rich
- Signals (we don't provide signals, we provide analysis)
- Advice, recommendations (legal reasons - we're a tool, not an advisor)
- Complex jargon without explanation
